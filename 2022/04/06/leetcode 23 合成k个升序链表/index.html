leetcode 23 合成K个升序链表

四月 06, 2022

题目

给你一个链表数组，每个链表都已经按升序排列。

请你将所有链表合并到一个升序链表中，返回合并后的链表。

示例 1：

输入：lists = [[1,4,5],[1,3,4],[2,6]]
输出：[1,1,2,3,4,4,5,6]
解释：链表数组如下：
[
1->4->5,
1->3->4,
2->6
]
将它们合并到一个有序链表中得到。
1->1->2->3->4->4->5->6
示例 2：

输入：lists = []
输出：[]
示例 3：

输入：lists = [[]]
输出：[]

提示：

k == lists.length
0 <= k <= 10^4
0 <= lists[i].length <= 500
-10^4 <= lists[i][j] <= 10^4
lists[i] 按 升序 排列
lists[i].length 的总和不超过 10^4

题解

[[1,4,5],[1,3,4],[2,6]]

升序合并

两个链表合并 -> 一个一个来, 合并

分治合并

相对于升序合并, 原始的两两合并会导致其中一个过大, 消耗过多时间

优先队列合并

首先所有的list都放到队列中

然后队列遍历, 当前最小的以next的形式加入到队列, 并加到合并的位置上

如果next = null 则不加入队列中
